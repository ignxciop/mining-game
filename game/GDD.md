# Project: Mining

## Concepto General

Juego incremental/idle inspirado en Keep on Mining!.

El jugador comienza con herramientas básicas y debe recolectar recursos,
mejorar su equipo, desbloquear automatizaciones y avanzar hacia minería
cada vez más eficiente.

El enfoque principal del juego es:

- progresión constante
- satisfacción visual
- automatización
- crecimiento exponencial
- árboles de habilidades
- optimización de builds

---

# Gameplay Loop

El gameplay principal consiste en:

1. Minar recursos
2. Obtener monedas/materiales
3. Comprar mejoras
4. Aumentar producción
5. Desbloquear automatizaciones
6. Reiniciar mediante prestige
7. Repetir con multiplicadores mayores

---

# Cómo se juega

El jugador puede:

- minar manualmente
- desbloquear minería automática
- mejorar herramientas
- desbloquear nuevas zonas
- conseguir minerales raros
- desbloquear perks
- especializar builds

El juego debe poder jugarse:

- activamente

---

# Objetivo del jugador

El objetivo es:

- maximizar producción
- desbloquear todas las zonas
- obtener herramientas legendarias
- optimizar builds
- progresar infinitamente

No existe un final real.
El progreso es potencialmente infinito.

# Sistemas principales

## Sistema de minería

Cada bloque tiene:

- HP
- rareza
- loot table
- dificultad

## Sistema de daño

El daño depende de:

- herramienta
- upgrades
- multiplicadores
- habilidades

## Sistema idle

La minería continúa automáticamente.

---

# Progression

El jugador desbloquea:

- nuevas herramientas
- nuevas zonas
- nuevos minerales
- automatizaciones
- perks
- sistemas avanzados

La progresión debe sentirse:

- rápida al inicio
- estratégica en midgame
- exponencial en lategame

---

# Prestige / Rebirth

El jugador puede reiniciar progreso para obtener:

- multiplicadores permanentes
- perks globales
- nuevas mecánicas

Cada prestige debe acelerar futuras runs.

---

# Árbol de habilidades

Ejemplos:

- +10% velocidad minería
- x2 minerales raros
- críticos
- minería explosiva
- drones automáticos

El árbol debe permitir builds distintas.
